# Como executar o projeto
Pré-requisitos: npm / yarn

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/danilolan/crud

# instalar dependências
npm install

#entrar na pasta frontend
cd frontend

# executar o projeto
npm start
```
